# Site Abaya

A beautiful and elegant website showcasing abaya fashion and collections.

##  Overview

Site Abaya is a responsive web application designed to present abaya styles, collections, and designs. The site combines modern web design with a focus on aesthetic presentation.

##  Technologies

This project is built with:

- **HTML** - Semantic markup and page structure
- **CSS** - Styling, layouts, and responsive design
- **JavaScript** - Interactivity and dynamic functionality

##  Features

- Responsive design for mobile, tablet, and desktop devices
- Clean and elegant user interface
- Showcase of abaya collections
- Interactive elements for better user engagement
